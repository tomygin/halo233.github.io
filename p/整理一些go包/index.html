      package main 可以拆分，且包名与文件名无关
比如有一下两个文件

A.go

package main
import "fmt"
func main(){
    a()
    b()
}

func a(){
    fmt.Println("我在A文件的main包里面")
}


B.go

package main

func b(){
    fmt.Println("我在B文件的main包里面")
}


如何运行完整的main包？
在终端输入
go run  A.go B.go
如果是Linux下直接 go run *.go

      go项目的目录情况应该这样


filename	//整个项目名称
	main.go
	go.mod
	go.sum //这个是go所需要的包的版本哈希值
	A	//A包
		name1.go	//所有文件内部都必须注明  package A 
		name2.go	//文件名称无所谓
		name3.go
	B	//B包
		name1.go
		name2.go
		name3.go

在 mian.go 中调用包应当这样写
import "filename/A"
	以前的版本可以 "./A" 但是现在报错

如果要把 A包 归纳到 mian 包
就这样写 import ."filename/A"
